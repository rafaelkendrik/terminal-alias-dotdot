DOTDOT
======

A simple custom alias to back directory navigation in terminal.

**Command Usage**:

| Description       | Usage       |
| :---------------- | :---------- |
| Back dir          | `..`        |
| Back N dir(s)     | `.. N`      |
| Go to $HOME       | `.. ~`      |
| Back to last dir  | `.. -`      |
| Go to ALIAS dir   | `.. ALIAS`  |

**Install**:

1 - Clone `.dotdot.sh` file:

```sh
$ wget https://raw.githubusercontent.com/rafaelkendrik/terminal-dotdot/master/.dotdot.sh
```

2 - Source `.dotdot.sh`:

```sh
$ source .dotdot.sh
```
